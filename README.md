# A.P.D.E
Apex predator dynamic ecosystem, or A.P.D.E for short is just a fun kinda wolfpack bit life type of thing, have fun!
More info
A text-based ecological simulation built in Python.

## About
Manage an endangered wolf pack by balancing prey populations,
surviving environmental disasters, and making strategic decisions
to keep your ecosystem healthy.

## Features
- 🐺 Wolf Pack Management
- 🌲 Habitat Conservation
- 🌦️ Seasonal Events
- 🌪️ Random Environmental Disasters
- 🎮 Interactive Decision Making
